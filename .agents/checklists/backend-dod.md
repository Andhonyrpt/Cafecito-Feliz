# Backend Definition of Done

- [ ] Existe spec aprobado.
- [ ] El cambio pertenece a una sola unidad de trabajo.
- [ ] ESM e imports locales con `.js`.
- [ ] Validadores y `validate` aplicados en rutas.
- [ ] Controladores delegan errores inesperados con `next(error)`.
- [ ] Admin routes mantienen `authMiddleware` antes de `isAdmin`.
- [ ] Tests Jest/Supertest/unitarios relevantes ejecutados.
- [ ] Contratos API documentados si cambian.
- [ ] No hay secrets ni datos sensibles en logs.
- [ ] No hay cambios no relacionados.
