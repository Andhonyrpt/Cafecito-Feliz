# Frontend Definition of Done

- [ ] Existe spec aprobado.
- [ ] El cambio pertenece a una sola unidad de trabajo.
- [ ] Se preserva el flujo POS real.
- [ ] No se agregan rutas e-commerce inexistentes.
- [ ] Servicios usan `src/services/http.js`.
- [ ] Estado local/remoto documentado si cambia.
- [ ] Tests frontend relevantes ejecutados o justificación documentada.
- [ ] No hay librerías nuevas sin justificación.
- [ ] No hay `console.log` debug ni código temporal.
- [ ] Documentación actualizada si cambia flujo.
