# PR Checklist Multiagente

- [ ] Pendiente proviene del backlog aprobado.
- [ ] Spec existe y referencia el pendiente.
- [ ] Rama creada desde `develop` actualizado.
- [ ] Se cumple `1 pendiente = 1 spec = 1 rama = 1 PR`.
- [ ] Implementador no se autoaprobó.
- [ ] CAs cumplidos/no cumplidos documentados.
- [ ] Tests ejecutados con evidencia.
- [ ] Security review ejecutada si aplica.
- [ ] Docs actualizadas si aplica.
- [ ] Pendientes nuevos registrados como propuesta/backlog.
- [ ] Orchestrator validó integración.
